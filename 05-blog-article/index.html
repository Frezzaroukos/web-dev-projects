10 HTML Tips για Αρχάριους Web Developers

Συγγραφέας:Άγγελος Φραντζεσκάκης
Δημοσίευση:29 Σεπτεμβρίου 2025
Χρόνος ανάγνωσης:8 λεπτά
Κατηγορία: Web Development, HTML

Εισαγωγή

Όταν ξεκίνησα να μαθαίνω web development, έκανα πολλά λάθη που θα μπορούσαν να είχαν αποφευχθεί. Σε αυτό το άρθρο, μοιράζομαι 10 tips που θα βοηθήσουν αρχάριους developers να γράφουν καλύτερο HTML κώδικα.

1. Χρησιμοποίησε Semantic HTML Tags

Αντί να χρησιμοποιείς <div> για τα πάντα, χρησιμοποίησε semantic tags που δίνουν νόημα στο περιεχόμενο.

Κακό Παράδειγμα:

<div class="header">
    <div class="navigation">...</div>
</div>
<div class="main">
    <div class="article">...</div>
</div>

Καλό Παράδειγμα:

<header>
    <nav>...</nav>
</header>
<main>
    <article>...</article>
</main>

Γιατί είναι σημαντικό: Semantic HTML βελτιώνει το SEO, την accessibility, και κάνει τον κώδικα πιο readable.

2. Πάντα Προσθέτε Alt Text στις Εικόνες

Τοalt attribute είναι απαραίτητο για το accessibility και SEO

Παράδειγμα:

<!-- ❌ Λάθος -->
<img src="logo.png">

<!-- ✅ Σωστό -->
<img src="logo.png" alt="Company Logo">

Tip: Γράψε descriptive alt text που εξηγεί τι δείχνει η εικόνα, όχι "image" ή "photo".

3. Χρησιμοποίησε Σωστές Heading Levels

Τα headings πρέπει να ακολουθούν ιεραρχική σειρά: <h1> → <h2> → <h3>

Κανόνες:

  1. Μόνο ένα <h1> ανά σελίδα
  2. Μην παραλείπεις επίπεδα (π.χ. από h2 σε h4)
  3. Χρησιμοποίησε headings για δομή, όχι για styling

4. Validation και Testing

Πάντα να ελέγχεις τον κώδικα σου για errors.Χρησιμοποίησε το W3C Markup Validator

Keyboard Shortcuts που βοηθάνε:

Ctrl + S
Save file
Ctrl + Shift + I
Άνοιγμα Developer Tools
F5
Refresh page
Ctrl + U
View page source

5. Indentation και Formatting

Καθαρός κώδικας είναι readable κώδικας. Χρησιμοποίησε consistent indentation.

Παράδειγμα με 2 spaces indentation:

<section> 
    <h2>Title</h2>
    <p>Content...</p>
    <ul>
        <li>Item 1</li>
        <li>Item 2</li>
    </ul>
</section>

VS Code Tip: Χρησιμοποίησε ένα code editor που υποστηρίζει auto-formatting, όπως το VSCode με Shift + Alt + F.

6. Meta Tags για SEO

Πρόσθεσε σωστά meta tags στο <head> για καλύτερο SEO.

Essential meta tags:

<meta charset="UTF-8">
<meta name="viewport" content="width=device-width, initial-scale=1.0">
<meta name="description" content="Page description">
<meta name="author" content="Your Name">
<title>Page Title</title>

Comments για Οργάνωση

Χρησιμοποίησε HTML comments για να οργανώσεις τον κώδικά σου.

Παράδειγμα:

<!-- Header Section -->
<header>...</header>

<!-- Main Content -->
<main>...</main>

Προσοχή: Μην βάζεις sensitive information σε comments - είναι ορατά στο source code!

8. Links και Navigation

Για external links, χρησιμοποίησε target="_blank"και rel="noopener noreferrer" για ασφάλεια.

Παράδειγμα:

<!--Internal link-->
<a href="/about.html">About</a>

<!--External link-->
<a href="https://example.com" target="_blank" rel="noopener noreferrer">Example</a>

9. Forms και User Input

Χρησιμοποίησε proper form elements με label tages για accessibility.

Παράδειγμα:

<form>
    <label for="email">Email:</label>
    <input type="email" id="email" name="email" required>

    <label for="message">Message:</label>
    <textarea id="message" name="message" required></textarea

    <button type="submit">Send</button>
    </form>

10. Testing σε Διαφορετικά Browsers

Ο κώδικάς σου πρέπει να δουλεύει σε όλα τα browsers. Test σε:

  • Chrome / Edge
  • Firefox
  • Safari (αν έχεις Mac)
  • Mobile browsers

Developer Tools tip:Πάτα F12 και χρησιμοποίησε το mobile device simulator

Συμπέρασμα

Αυτά τα 10 tips θα σε βοηθήσουν να γράφεις καλύτερο HTML κώδικα από την αρχή. Θυμήσου: η πρακτική κάνει τον δάσκαλο. Όσο περισσότερα projects φτιάχνεις, τόσο καλύτερος θα γίνεσαι.
